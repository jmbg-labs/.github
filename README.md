## About JMBG PHP Library

JMBG PHP Library is a small open-source project created to validate, parse and determine gender, age and region from Serbian unique master citizen numbers (JMBG - Jedinstveni Matični Broj Građana). 

### Features

- ✅ Validation
- 📊 Parsing
- 🎂 Determine age
- 👤 Determine gender
- 📍 Determine birth region
